Arch Linux AUR Installation (you can use kerminal-bin for binary): 


git clone https://aur.archlinux.org/kerminal.git
cd kerminal
makepkg -si

# Or use an AUR helper like yay:
yay -S kerminal

Or build from source:

git clone https://github.com/klpod221/kerminal.git
cd kerminal
npm install
npm run tauri build
