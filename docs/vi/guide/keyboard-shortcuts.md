# Phím Tắt

Kerminal cung cấp hệ thống phím tắt toàn diện để tăng năng suất làm việc. Tất cả phím tắt đều có thể tùy chỉnh trong Cài đặt.

::: tip Người dùng macOS
Trên macOS, `Ctrl` được thay bằng `Cmd (⌘)` cho tất cả phím tắt.
:::

## Thao Tác Terminal

| Hành động | Phím tắt | Mô tả |
|-----------|----------|-------|
| Tab mới | `Ctrl+T` | Mở tab terminal mới |
| Đóng tab | `Ctrl+W` | Đóng tab hiện tại |
| Tab tiếp theo | `Ctrl+Tab` | Chuyển sang tab kế tiếp |
| Chuyển đến Tab 1 | `Ctrl+1` | Nhảy trực tiếp đến tab 1 |
| Chuyển đến Tab 2 | `Ctrl+2` | Nhảy trực tiếp đến tab 2 |
| Chuyển đến Tab 3 | `Ctrl+3` | Nhảy trực tiếp đến tab 3 |
| Chuyển đến Tab 4-9 | `Ctrl+4` đến `Ctrl+9` | Nhảy đến tab 4-9 |
| Chia dọc | `Ctrl+K` | Chia terminal theo chiều dọc (cạnh nhau) |
| Chia ngang | `Ctrl+L` | Chia terminal theo chiều ngang (trên/dưới) |
| Tìm lịch sử | `Ctrl+H` | Tìm kiếm trong lịch sử lệnh |

## Điều Hướng Panel

| Hành động | Phím tắt | Mô tả |
|-----------|----------|-------|
| Focus Panel tiếp | `Ctrl+→` | Di chuyển focus sang panel kế (phải/dưới) |
| Focus Panel trước | `Ctrl+←` | Di chuyển focus về panel trước (trái/trên) |

## Truy Cập Nhanh

| Hành động | Phím tắt | Mô tả |
|-----------|----------|-------|
| Mở Workspace | `` Ctrl+` `` | Bật/tắt workspace terminal chính |
| Mở SSH Profiles | `Ctrl+Shift+S` | Mở ngăn kéo SSH profiles |
| Mở File Browser (SFTP) | `Ctrl+Shift+F` | Mở trình duyệt file SFTP |
| Mở Saved Commands | `Ctrl+Alt+C` | Mở ngăn kéo lệnh đã lưu |
| Command Palette | `Ctrl+Shift+P` | Mở bảng lệnh nhanh |

## Trợ Giúp

| Hành động | Phím tắt | Mô tả |
|-----------|----------|-------|
| Bắt đầu Tour | `Ctrl+Shift+/` | Khởi động tour hướng dẫn tính năng |

## Phím Tắt Trong Terminal

Các phím tắt này hoạt động trong terminal emulator:

| Hành động | Phím tắt | Mô tả |
|-----------|----------|-------|
| Sao chép | `Ctrl+Shift+C` | Sao chép văn bản đã chọn |
| Dán | `Ctrl+Shift+V` | Dán từ clipboard |
| Chọn tất cả | `Ctrl+Shift+A` | Chọn toàn bộ nội dung terminal |
| Xóa Terminal | `Ctrl+Shift+K` | Xóa bộ đệm terminal |
| Tìm trong Terminal | `Ctrl+Shift+F` | Mở hộp thoại tìm kiếm |
| Phóng to | `Ctrl++` | Tăng cỡ chữ |
| Thu nhỏ | `Ctrl+-` | Giảm cỡ chữ |
| Đặt lại Zoom | `Ctrl+0` | Khôi phục cỡ chữ mặc định |

## Command Palette

Command Palette (`Ctrl+Shift+P`) cho phép truy cập nhanh mọi hành động:

1. Nhấn `Ctrl+Shift+P` để mở
2. Bắt đầu gõ để lọc lệnh
3. Dùng phím `↑` / `↓` để điều hướng
4. Nhấn `Enter` để thực thi lệnh đã chọn
5. Nhấn `Escape` để đóng

## Tùy Chỉnh Phím Tắt

Bạn có thể tùy chỉnh bất kỳ phím tắt nào:

1. Vào **Cài đặt** → **Phím tắt**
2. Nhấp vào phím tắt muốn thay đổi
3. Nhấn tổ hợp phím mong muốn
4. Nhấp **Lưu** để áp dụng
5. Nhấp **Đặt lại** để khôi phục mặc định

::: warning Phát Hiện Xung Đột
Kerminal sẽ cảnh báo nếu phím tắt tùy chỉnh của bạn xung đột với phím tắt hiện có.
:::

## Mẹo Cho Người Dùng Chuyên Nghiệp

### Chuyển Tab Nhanh

- Dùng `Ctrl+1` đến `Ctrl+9` để truy cập tab ngay lập tức
- `Ctrl+Tab` xoay vòng qua các tab theo thứ tự
- `Ctrl+W` đóng tab hiện tại ngay lập tức

### Quản Lý Panel Hiệu Quả

- `Ctrl+K` → `Ctrl+K` → `Ctrl+K` tạo bố cục 3 cột
- `Ctrl+L` → `Ctrl+L` tạo bố cục 3 hàng
- Kết hợp chia để tạo bố cục lưới phức tạp

### Điều Hướng Nhanh

- Luôn dùng Command Palette (`Ctrl+Shift+P`) cho mọi hành động
- Dùng `Ctrl+Arrow` để điều hướng giữa các panel chia
- `` Ctrl+` `` bật/tắt workspace nhanh chóng
