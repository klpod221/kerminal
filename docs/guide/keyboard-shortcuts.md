# Keyboard Shortcuts

Kerminal provides comprehensive keyboard shortcuts to enhance your productivity. All shortcuts are fully customizable via Settings.

::: tip macOS Users
On macOS, `Ctrl` is replaced with `Cmd (⌘)` for all shortcuts.
:::

## Terminal Operations

| Action | Shortcut | Description |
|--------|----------|-------------|
| New Tab | `Ctrl+T` | Open a new terminal tab |
| Close Tab | `Ctrl+W` | Close the current tab |
| Next Tab | `Ctrl+Tab` | Switch to the next tab |
| Switch to Tab 1 | `Ctrl+1` | Jump directly to tab 1 |
| Switch to Tab 2 | `Ctrl+2` | Jump directly to tab 2 |
| Switch to Tab 3 | `Ctrl+3` | Jump directly to tab 3 |
| Switch to Tab 4-9 | `Ctrl+4` to `Ctrl+9` | Jump to tabs 4-9 |
| Split Vertically | `Ctrl+K` | Split terminal vertically (side by side) |
| Split Horizontally | `Ctrl+L` | Split terminal horizontally (top/bottom) |
| Search History | `Ctrl+H` | Search through command history |

## Panel Navigation

| Action | Shortcut | Description |
|--------|----------|-------------|
| Focus Next Panel | `Ctrl+→` | Move focus to the next panel (right/down) |
| Focus Previous Panel | `Ctrl+←` | Move focus to the previous panel (left/up) |

## Quick Access

| Action | Shortcut | Description |
|--------|----------|-------------|
| Open Terminal Workspace | `` Ctrl+` `` | Toggle the main terminal workspace |
| Open SSH Profiles | `Ctrl+Shift+S` | Open SSH profiles drawer |
| Open File Browser (SFTP) | `Ctrl+Shift+F` | Open the SFTP file browser |
| Open Saved Commands | `Ctrl+Alt+C` | Open saved commands drawer |
| Command Palette | `Ctrl+Shift+P` | Open the command palette |

## Help & Tour

| Action | Shortcut | Description |
|--------|----------|-------------|
| Start Feature Tour | `Ctrl+Shift+/` | Start the interactive feature tour |

## Terminal-Specific Shortcuts

These shortcuts work within the terminal emulator:

| Action | Shortcut | Description |
|--------|----------|-------------|
| Copy | `Ctrl+Shift+C` | Copy selected text |
| Paste | `Ctrl+Shift+V` | Paste from clipboard |
| Select All | `Ctrl+Shift+A` | Select all terminal content |
| Clear Terminal | `Ctrl+Shift+K` | Clear terminal buffer |
| Find in Terminal | `Ctrl+Shift+F` | Open find dialog (when not in SFTP) |
| Zoom In | `Ctrl++` | Increase font size |
| Zoom Out | `Ctrl+-` | Decrease font size |
| Reset Zoom | `Ctrl+0` | Reset to default font size |

## Command Palette

The Command Palette (`Ctrl+Shift+P`) provides quick access to all actions:

1. Press `Ctrl+Shift+P` to open
2. Start typing to filter commands
3. Use `↑` / `↓` arrow keys to navigate
4. Press `Enter` to execute the selected command
5. Press `Escape` to close

## Customizing Shortcuts

You can customize any keyboard shortcut:

1. Go to **Settings** → **Keyboard Shortcuts**
2. Click on the shortcut you want to change
3. Press your desired key combination
4. Click **Save** to apply changes
5. Click **Reset** to restore defaults

::: warning Conflict Detection
Kerminal will warn you if your custom shortcut conflicts with an existing one.
:::

## Tips for Power Users

### Quick Tab Switching

- Use `Ctrl+1` through `Ctrl+9` for instant tab access
- `Ctrl+Tab` cycles through tabs in order
- `Ctrl+W` closes the current tab instantly

### Efficient Panel Management

- `Ctrl+K` → `Ctrl+K` → `Ctrl+K` creates a three-column layout
- `Ctrl+L` → `Ctrl+L` creates a three-row layout
- Combine splits for complex grid layouts

### Rapid Navigation

- Keep command palette (`Ctrl+Shift+P`) handy for any action
- Use `Ctrl+Arrow` keys to navigate between split panels
- `` Ctrl+` `` toggles the workspace quickly
