# Getting Started

Welcome to **Kerminal** - a modern, high-performance terminal emulator that combines the power of a full-featured local terminal with advanced SSH connection management.

## Quick Overview

### Terminal Features

| Feature | Description |
|---------|-------------|
| Multiple Tabs | Work with multiple terminals simultaneously |
| Split Panes | Divide your workspace horizontally or vertically |
| WebGL Rendering | Hardware-accelerated terminal rendering |
| Unicode Support | Full Unicode 11 character support |

### SSH Features

| Feature | Description |
|---------|-------------|
| Profile Management | Organize connections with groups and colors |
| Key Authentication | Support for SSH keys, certificates, and more |
| Port Forwarding | Local, remote, and dynamic SOCKS tunneling |
| Jump Hosts | Chain through multiple bastion hosts |
